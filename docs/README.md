# Documentation Index

This folder contains all project documentation for the Telegram Payment Gateway.

## Structure

- `guides/` — Integration guides, API usage, developer onboarding
- `plans/` — Completion plans, phase plans, project roadmap
- `reference/` — Architecture, API reference, settlement flow, smart contract docs
- `setup/` — Deployment, environment setup, Render config
- `legacy/` — Old/archived docs, cleanup summaries
- `archive/` — Dashboard and deprecated plans
- `reports/` — Test results, lint reports, deployment readiness
- `security/` — Incident reports and urgent actions
- `process/` — Contributing and process notes
- `reviews/` — Code review responses and review artifacts
